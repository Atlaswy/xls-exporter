excel配置表导出工具<br>

# 特点
- 详细的报错
- 支持列约束
- 支持嵌套数据定义
- 严格类型检查
- 支持自定义Formatter输出其他语法
- 同时兼容py2/py3
- 保持源数据顺序输出
- 支持注释行
- 支持多sheet
- 三种结构模式:定义、列表、字典

# 内部执行流程
```
xls file->TypeTree->ValueTree-->LuaFormatter->lua file
                             |->JsonFormatter->json file
                             |->...

```

# 基础数据类型
- int
- float
- bool
- string

# 嵌套数据定义
- 字典
```
--------------------
|int     |string   |
--------------------
|user.age|user.name|
--------------------
|14      |haha     |
--------------------
生成:
user={age=14,name="haha"}
```

- 数组
```
-----------------------------------------------------
|int         |int         |int         |int         |
-----------------------------------------------------
|skills[0].cd|skills[0].id|skills[1].cd|skills[1].id|
-----------------------------------------------------
|1           |2           |3           |4           |
-----------------------------------------------------
生成:
skills=[{cd=1,id=2},{cd=3,id=4}]
```

- 内嵌数组
```
---------------
|int[]        |
---------------
|skills       |
---------------
|1,2,3        |
---------------
生成：
skills=[1,2,3]
```
